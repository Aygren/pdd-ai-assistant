# 🚘 PDD Assistant

Интеллектуальный ИИ-помощник для анализа обстоятельств ДТП и юридической консультации на основе актуальных Правил дорожного движения РФ.

## 🔥 Ключевая особенность (Архитектура)

В отличие от стандартных RAG-систем, проект реализует паттерн **Information Gathering Agent** (Квалифицирующий агент):
1. **Сбор фактов:** Модель не гадает, а проверяет ситуацию по чек-листу ГИБДД через структурированный JSON-анализ.
2. **Дорожная логика:** ИИ понимает иерархию правил (например, что работающий светофор отменяет действие знаков приоритета).
3. **Уточняющие вопросы:** Если данных для вердикта мало, бот сам задает точечные вопросы водителю в чате.
4. **Гибридный RAG:** Финальный юридический разбор формируется только после сбора 100% фактов на основе поиска по базе данных.

## 🛠 Стек технологий

* **Язык:** Python 3.12
* **Фреймворк:** LangChain
* **LLM:** Mistral AI (`mistral-small-latest`) c поддержкой JSON-mode
* **База знаний (Vector Store):** Supabase (pgvector)
* **Интерфейс:** Streamlit

## 🚀 Инструкция по развертыванию, настройке и запуску

Для работы с проектом выполните последовательно следующие команды в терминале:

```bash
# 1. Создание виртуального окружения
python -m venv .venv

# 2. Активация окружения (для Linux/macOS)
source .venv/bin/activate

# 3. Активация окружения (для Windows)
# .venv\Scripts\activate

# 4. Установка необходимых библиотек
pip install -r requirements.txt

# 5. НАСТРОЙКА ОКРУЖЕНИЯ (.env)
# Создайте файл .env в корне проекта и добавьте ваши ключи:
# MISTRAL_API_KEY=your_mistral_api_key_here
# SUPABASE_URL=your_supabase_project_url_here
# SUPABASE_SERVICE_KEY=your_supabase_service_role_key_here

# 6. ЗАПУСК ПРИЛОЖЕНИЯ
streamlit run app.py
